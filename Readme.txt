**Setup**

Only works on Python 3.4+.

Just install `requirements.txt` and then `python run.py`.

To read the annotated source code, start with run.py. You might want to keep
this page handy while you do: http://steveasleep.com/clubsandwich/

For binaries and additional info, check out the Ludum Dare page for the game:
https://ldjam.com/events/ludum-dare/38/rogue-basement
